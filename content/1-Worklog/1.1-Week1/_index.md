### Week 1 Achievements:

* Connected and became acquainted with First Cloud AI Journey team members.

* Understood what AWS is and mastered the basic service groups: 
  * Compute (EC2, Lambda, ECS, etc.)
  * Storage (S3, EBS, EFS, etc.)
  * Networking (VPC, Route 53, CloudFront, etc.)
  * Database (RDS, DynamoDB, Aurora, etc.)
  * Management & Governance
  * Security, Identity & Compliance

* Successfully created and configured an AWS Free Tier account.

* Became familiar with the AWS Management Console and learned how to navigate, find, and use AWS services.

* Installed and configured AWS CLI on the computer, including:
  * Access Key and Secret Key configuration
  * Default Region setup
  * Output format configuration

* Used AWS CLI to perform basic operations and verified the setup.

* Learned about AWS Budgets and cost management:
  * Understanding different budget types (Cost, Usage, Reservation, Savings Plans)
  * How to set up budget alerts and notifications
  * Best practices for monitoring AWS spending
  * Cost optimization strategies

* Successfully created and configured AWS Budgets to track and monitor spending.

* Mastered IAM (Identity and Access Management) fundamentals:
  * Understanding Users, Groups, and Roles
  * IAM Policies and permission management
  * Policy structure and JSON syntax
  * Multi-Factor Authentication (MFA) setup
  * Security best practices and principle of least privilege

* Practiced creating IAM users and groups with appropriate policies.

* Configured MFA to enhance account security.

* Acquired foundational knowledge to manage AWS resources securely and cost-effectively.
